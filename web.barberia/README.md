# Golden Fade

Landing page moderna y responsive de una barbería premium desarrollada con HTML, CSS y JavaScript.

## Características
- Diseño responsive
- Navbar dinámica
- Animaciones al hacer scroll
- Loader inicial
- Botón flotante de WhatsApp
- Diseño luxury negro y dorado
- Optimización móvil

## Tecnologías
- HTML5
- CSS3
- JavaScript
